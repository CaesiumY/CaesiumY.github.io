이번에 설치할 목록은 다음과 같다.

  1. Visual Studio Code
  2. git
  3. python
  4. django


vscode 설치하기

vscode 로고

vscode 설치 링크 에 들어가서 윈도우OS 전용으로 다운 받은 뒤, 파일을 실행하면 빠른 설치가 가능하다.

…면 좋겠으나, 막 누르지는 말고 아래 사진이 나오면 스탑!!

체크 버튼이 나오는 항목까지만 진행하자
