- **Next.js의 렌더링 원리 이해**: Server Component와 SSR, CSR의 차이를 명확히 이해하고, 페이지 성격에 따라 렌더링 전략을 최적화하여 적용합니다.

### Tools

- **생산성 도구 기반 DX 개선**: Orval과 같은 코드 제너레이터를 도입해 반복 코딩을 줄이고 팀 전체의 개발자 경험(DX)을 혁신합니다.
- **툴 마스터**: 에디터와 브라우저의 업데이트 릴리즈와 유용한 확장 프로그램을 끊임없이 발굴하고, 이를 동료들에게 전파하여 팀의 능률을 높입니다.

### UI/UX

- **디테일한 사용자 경험**: 긍정적인 마이크로카피(Microcopy) 한 줄이 서비스의 인상을 좌우한다고 믿습니다. 기술적 구현을 넘어 유저와의 소통까지 고민합니다.
- **사용자 관점의 문제 해결**: 개발 과정에서 유저가 느낄 수 있는 미세한 불편함(Latency, Layout Shift 등)을 예민하게 포착하고 선제적으로 개선합니다.

### AI Agents

- **AI-Native Development**: Claude Code와 Cursor 등 AI 에이전트를 개발 파이프라인의 핵심으로 활용해 생산성의 한계를 극복하는 방법을 연구합니다.
- **바이브 코딩이 아닌 협업 코딩**: Claude Code(MAX 200), Cursor Pro 등 고성능 모델에 아낌 없이 투자하고 활용하며, 적극적으로 오케스트레이션(Orchestration)합니다.
