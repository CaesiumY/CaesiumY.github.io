# 자랑하고 싶은 개발자, 윤창식입니다.

새로 만나는 기술에 빠르게 적응하고, 여러 도구를 이용한 최적화에 집중하며, 공유와 오픈 소스를 통한 성장에 열정적인 **프론트엔드 개발자**입니다.

## 현재

AI를 활용해 외주 프로젝트를 동시에 진행하며, **AI Agent와 최신 기술 스택**을 활용한 효율적인 개발 방법론을 탐구하고 있습니다. 특히 [교육 지원 플랫폼](https://www.chungbook-e.com/)과 [한국 기술 마켓](https://kotechmarket.com/) 프로젝트에서 모던 프론트엔드 기술과 AI 도구를 적극 활용하고 있습니다.

---

## 주요 경험

### 팀스파르타 튜터 (2023.12 ~ 2025.05)

**내배캠 리액트 트랙 튜터**로 풀타임(9to9) 상주하며 밀착 멘토링을 진행했습니다.

- 80회 이상의 맞춤형 라이브 강의를 진행하여 **중도 포기자 75% 감소** 달성
- 최신 기술 트렌드를 반영한 프론트엔드 커리큘럼 기획 및 강의 콘텐츠 제작
- 'VSCode 활용법' 등 반복 기술 질문 해결을 위한 실무 기술 특강 진행
- 최소 기술 스택 기반의 인턴십 과제 설계 및 변별력 있는 평가 지표 수립

### KC-MIC 프론트엔드 엔지니어 (2022.03 ~ 2023.09)

레거시 시스템 현대화부터 실시간 시뮬레이터까지 다양한 프론트엔드 개발을 담당했습니다.

- 레거시 프론트엔드 마이그레이션(Vite, RTK)을 주도하여 **빌드 속도 70% 단축**
- Next.js 서버 컴포넌트/액션을 중심으로 효율적인 렌더링 및 데이터 관리 아키텍처 설계
- SSR 환경에서 TanStack Query를 도입하여 데이터 정합성 및 UI 안정성 확보
- Phaser.js 기반 Scene 개발 및 gRPC/Protobuf를 활용한 실시간 다중 접속 시뮬레이터 UI 구현
- Notion API를 연동하여 비개발 직군을 위한 CMS 구축

## 외주 프로젝트

### [교육 지원 플랫폼](https://www.chungbook-e.com/) (2025.05 ~)

Next.js, shadcn/ui, Orval을 활용한 대시보드 및 유저 페이지 개발

- Orval 도입을 통한 Swagger 문서 기반 API 스키마 자동 생성
- MSW를 활용한 API Mocking으로 안정적인 테스트 환경 구축
- AI Agents(Cursor, Claude Code)를 적극 활용하여 **마감의 50% 지점에 목표 완수**

### [한국 기술 마켓](https://kotechmarket.com/) (2024.05 ~)

Next.js, Turborepo, shadcn/ui를 활용한 대시보드 및 유저 페이지 개발 ([kotechhub.com](https://kotechhub.com/))

- 관리자 대시보드 A to Z 설계 및 구현을 통해 프로젝트 초기 구축 주도
- 레거시 코드를 재설계하고, 전체 앱 리팩터링으로 코드 품질 및 유지보수성 개선
- Turborepo 기반 모노레포 아키텍처로 마이그레이션하여 프로젝트 확장성 확보
- FSD(Feature-Sliced Design) 구조와 디자인 토큰 시스템 도입

### [마치 무비 나잇](https://mrch-movie-night.vercel.app/) (2023.02)

Next.js, Firebase를 활용한 공연 홍보 및 예매용 웹 플랫폼

- 플랫폼 기획부터 프론트엔드, 백엔드(Firebase)까지 풀스택 프로덕트 라이프사이클 경험
- Firebase 기반 실시간 좌석 예매 시스템 및 예매자 정보 관리용 대시보드 구현

### [노션 API 블로그](https://github.com/CaesiumY/notion-blog-v2) (2022.10 ~ 2023.05)

Next.js와 TypeScript로 만드는 NotionAPI 블로그 강의 자료

- Pages Router에서 App Router로의 마이그레이션을 통한 최신 아키텍처 적용
- 빌드 프로세스 내 게시글 썸네일 자동 생성 기능 구현

### [마헤 뮤지컬 사이트](https://musical-ticketing.vercel.app/) (2022.04)

Next.js, TailwindCSS를 활용한 뮤지컬 홍보 및 예매 사이트

- 웹 폰트 SVG 변환을 통한 초기 로딩 속도 개선 및 FCP 단축
- **Lighthouse 모든 항목 100점 달성**을 통한 성능 최적화

## 기술 역량

### React / Next.js

- Next.js의 서버 컴포넌트와 서버 사이드 렌더링의 차이점을 이해하고 활용합니다.
- 커스텀 훅을 통해 UI와 비즈니스 로직을 분리하여 올바른 컴포넌트 설계를 추구합니다.

### 개발 도구

- Orval 같은 생산성 도구를 적극적으로 도입해 개발자 경험을 개선합니다.
- 에디터와 브라우저의 확장 프로그램을 적극 활용하고 추천합니다.

### UI/UX

- 긍정적인 마이크로카피 문구를 통한 최고의 사용자 경험을 추구합니다.
- 앱 사용 시에 유저가 느낄 불편함을 섬세하고 빠르게 캐치합니다.

### AI Agents

- AI 개발 도구(Claude Code, Cursor 등)를 적극 활용해 개발 생산성을 극대화하는 방법을 탐구합니다.
- Claude Code(MAX 200), Cursor Pro를 구독 중입니다.
