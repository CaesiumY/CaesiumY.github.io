# /translate-blog

이 커맨드는 `/translate-writer` 스킬을 호출하여 번역 작업을 수행합니다.

## 스킬이 제공하는 기능

- ✅ 스타일 가이드 및 용어집 참조
- ✅ 번역 품질 자동 검토 (10점 만점, 8점 이상 통과)
- ✅ 자동 수정 루프 (8점 미만 시 최대 3회)
- ✅ 사용자 최종 승인 단계
- ✅ 피드백 학습 및 스타일 가이드 자동 업데이트
- ✅ 새 용어 발견 시 용어집 업데이트 제안

## 실행

Skill 도구를 사용하여 `translate-writer` 스킬을 호출하세요.

**사용자 입력**: $ARGUMENTS
