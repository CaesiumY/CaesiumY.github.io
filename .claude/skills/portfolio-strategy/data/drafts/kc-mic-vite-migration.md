### KC-MIC · 프론트엔드 엔지니어 · 빌드 환경 현대화 (1년 6개월)

**문제**

- 당시 CRA가 eject로 웹팩 설정을 노출 + craco로 덮어씌운 비표준 환경이었다. 설정 추적이 어려웠고, 무엇보다 개발 서버·빌드 속도가 현저히 느려 DX에 직접 영향을 주고 있었다.

**해결**

- 검토한 두 대안은 (1) Vite 진보, (2) 웹팩 설정을 되돌려 순정 CRA로 회귀였다. 속도 이슈 해결을 우선 목표로 두고, 가장 빠르다고 평가받고 생태계에서 떠오르고 있던 Vite를 채택했다.
- Vite 마이그레이션 자체가 생각보다 쉬워 한 번에 진행했다. 누적된 craco 비표준 환경 정리가 동시에 이루어지면서 기술 부채를 함께 해소했다.
- 클래스 컴포넌트 → 함수형 전환은 Hook 유용성과 React 발전 방향을 근거로 팀 합의를 도출했다.

**결과**

- 빌드 시간 **70% 단축** (빌드 명령어 출력 기준 실측). 비표준 빌드 환경 정리로 DX 회복.
